## 참고: 삭제/수정 근거

### OS 최소 사양 상향 이유 (2026년 1월 기준)

**iOS 18 선정 근거**
- iOS 17: 2025년 9월 보안 지원 종료
- iOS 18: 2027년까지 보안 업데이트 지원 예정
- iOS 18 시장 점유율: 약 40~60% (2026년 1월 기준)
- 지원 기기: iPhone XS/XR (2018년) 이상

**Android 12 선정 근거**
- Android 11 이하: 보안 업데이트 종료
- Android 12+: 현재 활성 보안 지원 중
- Android 12+ 시장 점유율: 약 85% 이상
- 스마트폰 평균 교체 주기 3~4년 → 2022년 기기 기준

**하드웨어 기준**
- 2022년 중저가 스마트폰 스펙 기준
- RAM 4GB 이상, 저장공간 64GB 이상
- 대표 기기: iPhone 14, Galaxy S22, Pixel 6a 급

### Objective-C / Swift 삭제 이유
- Expo Managed Workflow에서 개발자가 직접 작성하지 않음
- Config Plugin은 JavaScript로 작성, 네이티브 설정 파일만 수정
- 빌드 시 자동 생성되는 네이티브 코드일 뿐 개발 언어 아님

### MCU 적용 해당 없음 이유
- MCU = Microcontroller Unit (마이크로컨트롤러)
- 임베디드 하드웨어 의료기기용 항목
- 모바일 앱/웹 소프트웨어에는 해당 없음
