## 1. IRB란 무엇인가?

### 1.1 정의

**IRB (Institutional Review Board)** = **기관생명윤리위원회**

인간 대상 연구(임상시험)의 윤리적 타당성을 심의하고 승인하는 독립 기관.

```
IRB의 핵심 역할:
┌────────────────────────────────────────────────────────────┐
│                                                            │
│   연구 참여자(피험자)의 권리, 안전, 복지 보호              │
│                                                            │
│   • 연구 윤리 적합성 검토                                  │
│   • 동의서(Informed Consent) 검토                          │
│   • 위험/이익 비율 평가                                    │
│   • 연구 진행 모니터링                                     │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

### 1.2 법적 근거

| 국가 | 규제 | 비고 |
|------|------|------|
| 한국 | 생명윤리 및 안전에 관한 법률 | 기관생명윤리위원회 운영 |
| 미국 | 21 CFR Part 50, 56 | FDA IRB 규정 |
| 국제 | ICH GCP (E6) | 국제임상시험가이드라인 |

### 1.3 IRB vs 기타 기관

| 기관 | 역할 | 관할 |
|------|------|------|
| **IRB** | 임상시험 윤리 심의 | 인체대상 연구 |
| **식약처(MFDS)** | 의료기기 인허가 | 제품 안전성/유효성 |
| **IACUC** | 동물실험 심의 | 동물대상 연구 |

## 2. GMP와 IRB의 관계

### 2.1 핵심 차이

```
┌─────────────────────────────────────────────────────────────────┐
│                    GMP vs IRB 비교                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   GMP (Good Manufacturing Practice)                             │
│   ─────────────────────────────────                             │
│   • 목적: 제품 품질 보증                                         │
│   • 대상: 제조/개발 프로세스                                     │
│   • 결과: 식약처 인허가                                          │
│   • 질문: "이 제품이 안전하고 품질이 보장되는가?"               │
│                                                                 │
│   IRB (Institutional Review Board)                              │
│   ─────────────────────────────────                             │
│   • 목적: 연구 참여자 보호                                       │
│   • 대상: 임상시험/인체대상 연구                                 │
│   • 결과: 임상시험 승인                                          │
│   • 질문: "이 연구가 윤리적으로 적절한가?"                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 진행 순서

**디지털치료제 일반적인 흐름:**

```
┌─────────────────────────────────────────────────────────────────┐
│                    DTx 인허가 프로세스                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Phase 1: 제품 개발                                            │
│   ├── 소프트웨어 개발                                           │
│   └── 비임상 검증 (기능 테스트)                                  │
│                                                                 │
│   Phase 2: GMP 문서 작성                                        │
│   ├── SRS (요구사항 명세서)                                      │
│   ├── SDF (상세 설계 파일)     ← 현재 진행중                    │
│   ├── STP/STR (테스트 문서)                                      │
│   └── RMP (위험관리)                                            │
│                                                                 │
│   Phase 3: 임상시험 (필요시)    ← IRB 필요                      │
│   ├── 임상시험 계획서 작성                                       │
│   ├── IRB 심의 및 승인                                          │
│   ├── 임상시험 실시                                             │
│   └── 임상 결과 보고서                                          │
│                                                                 │
│   Phase 4: 식약처 인허가 신청                                   │
│   ├── GMP 문서 제출                                             │
│   ├── 임상시험 결과 제출 (필요시)                               │
│   └── 기술문서 심사                                             │
│                                                                 │
│   Phase 5: 인허가 획득 → 시판                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 GMP 인허가 후 IRB?

**질문: "GMP 인허가 획득하고 IRB 신청하는 건가?"**

**답: 일반적으로 IRB가 먼저, GMP 인허가가 나중**

```
┌─────────────────────────────────────────────────────────────────┐
│                    일반적인 순서                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   임상시험이 인허가에 필요한 경우:                               │
│                                                                 │
│   1. GMP 문서 작성 (제품 품질 문서화)                            │
│   2. 임상시험 계획서 작성                                        │
│   3. IRB 승인 획득           ← IRB 먼저                         │
│   4. 임상시험 실시                                               │
│   5. 임상 결과 + GMP 문서로 인허가 신청                         │
│   6. 식약처 인허가 획득      ← 인허가 나중                       │
│                                                                 │
│   이유:                                                         │
│   • 임상시험 결과가 인허가 심사에 필요한 증거가 됨               │
│   • 임상시험 없이는 유효성 입증이 어려움 (특히 DTx)              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    예외적 경우                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   시판 후 추가 연구:                                             │
│                                                                 │
│   1. GMP 인허가 획득 → 시판                                     │
│   2. 추가 임상 연구 계획                                         │
│   3. IRB 승인 획득           ← 이 경우 인허가 후 IRB            │
│   4. 추가 연구 실시                                              │
│                                                                 │
│   이 경우는 이미 허가된 제품에 대한 추가 연구일 때               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. IRB가 필요한 경우

### 3.1 IRB 심의 필요

| 상황 | 필요 여부 |
|------|----------|
| 환자 대상 임상시험 | ✅ 필수 |
| 피험자 건강정보 수집 | ✅ 필수 |
| 앱 효과성 검증 연구 | ✅ 필수 |
| 사용자 설문조사 (개인정보 포함) | ✅ 필수 |

### 3.2 IRB 심의 불필요

| 상황 | 비고 |
|------|------|
| 내부 품질 테스트 | 인체 대상 아님 |
| 익명화된 기존 데이터 분석 | 개인식별 불가 |
| 기능 검증 (테스터 동의) | 연구 목적 아님 |

### 3.3 디지털치료제 특수성

```
DTx는 대부분 IRB 심의 필요

이유:
• 치료 효과 입증이 핵심 → 임상시험 필수
• 환자 데이터 수집/분석 → 인체대상연구
• 식약처 인허가에 임상 결과 요구
```

## 4. IRB 심의 절차

### 4.1 심의 단계

```
┌─────────────────────────────────────────────────────────────────┐
│                    IRB 심의 절차                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. 신청서 준비                                                 │
│      ├── 연구계획서 (Protocol)                                  │
│      ├── 동의서 (Informed Consent)                              │
│      ├── 설문지/평가도구                                        │
│      └── 연구자 자격 서류                                       │
│                                                                 │
│   2. IRB 접수                                                   │
│      └── 기관 IRB 또는 중앙 IRB에 제출                          │
│                                                                 │
│   3. 심의                                                       │
│      ├── 정기심의 (Full Board Review)                          │
│      └── 신속심의 (Expedited Review) - 저위험 연구              │
│                                                                 │
│   4. 결과                                                       │
│      ├── 승인 (Approval)                                        │
│      ├── 조건부 승인 (Conditional)                              │
│      ├── 보완 요청 (Modification Required)                      │
│      └── 불승인 (Disapproval)                                   │
│                                                                 │
│   5. 연구 실시                                                   │
│      └── 승인 후 연구 시작 가능                                  │
│                                                                 │
│   6. 지속심의                                                   │
│      └── 연 1회 이상 지속 심의                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 주요 제출 서류

| 서류 | 내용 |
|------|------|
| 연구계획서 | 연구 목적, 방법, 기간, 대상자 수 등 |
| 동의서 | 연구 참여 동의 양식 |
| 연구자 정보 | 연구책임자, 공동연구자 자격 |
| 위험/이익 분석 | 참여자 위험과 기대 이익 |
| 개인정보보호 계획 | 데이터 수집, 저장, 폐기 방법 |

## 5. 디지털치료제와 IRB

### 5.1 DTx 임상시험 특징

```
디지털치료제 임상시험의 특수성:

1. 비침습적
   • 소프트웨어 기반으로 물리적 위험 낮음
   • 그러나 심리적/행동적 영향 고려 필요

2. 데이터 수집
   • 앱 사용 로그, 건강 데이터 수집
   • 개인정보보호 철저히 해야 함

3. 대조군 설계
   • 플라시보 앱 vs 실제 치료 앱
   • 대기자 명단(Waitlist) 대조군

4. 평가 지표
   • 증상 개선 (통증 감소, 우울 감소 등)
   • 행동 변화 (복약 준수율 등)
   • 삶의 질 향상
```

### 5.2 ABCare 관련 임상 고려사항

| 기능 | 임상 평가 가능 지표 |
|------|-------------------|
| 복약관리 | 복약 준수율 향상 |
| 통증기록 | 통증 관리 개선, 삶의 질 |
| CBT/마음약국 | 불안/우울 감소, 대처 능력 |
| 습관점수 | 건강 행동 변화 |

### 5.3 IRB 신청 시 개발팀 역할

```
개발팀이 제공해야 할 정보:

1. 앱 기능 설명서
   • 각 기능이 어떻게 작동하는지
   • 사용자 인터페이스 스크린샷

2. 데이터 수집 명세
   • 수집되는 데이터 항목
   • 데이터 저장/전송 방식
   • 보안 조치 (암호화 등)

3. 위험 분석 정보
   • 앱 오류 시 가능한 위험
   • 대응 방안

4. 사용자 가이드
   • 연구 참여자용 앱 사용 설명
```

## 6. 자주 묻는 질문

### Q1: GMP 문서 작성과 IRB 신청을 병행할 수 있나?

**A: 가능. 실제로 많이 병행함.**

```
병행 가능한 이유:
• GMP 문서: 제품 자체의 품질/안전성 문서화
• IRB 신청: 임상시험 윤리 심의

두 작업은 독립적으로 진행 가능.
다만 IRB 신청 시 제품 안정 버전이 필요하므로,
개발이 어느 정도 완료된 후 IRB 신청하는 것이 일반적.
```

### Q2: IRB 승인 없이 식약처 인허가 가능한가?

**A: 경우에 따라 다름.**

```
IRB 없이 가능한 경우:
• 기존 임상 데이터로 인허가 (해외 데이터 등)
• 저위험 의료기기 (Class I)
• 문헌 근거만으로 충분한 경우

IRB 필요한 경우:
• 신규 임상시험 필요
• 국내 임상 데이터 요구
• 고위험 의료기기 (Class III)
• DTx는 대부분 임상 필요
```

### Q3: IRB 심의 기간은?

**A: 일반적으로 1-2개월**

```
심의 유형별 기간:
• 정기심의: 1-2개월 (월 1회 개최)
• 신속심의: 2-4주 (저위험 연구)
• 보완 시: 추가 1-2개월

※ 기관마다 다름, 여유 있게 계획 필요
```

### Q4: 어느 IRB에 신청해야 하나?

**A: 기관 IRB 또는 중앙 IRB**

```
기관 IRB:
• 병원, 대학 등 연구 수행 기관의 IRB
• 해당 기관에서 연구 수행 시

중앙 IRB:
• 독립적인 상업 IRB
• 다기관 연구, 기관 IRB 없을 때
• 예: 한국의료윤리심의위원회(KIRB)
```
